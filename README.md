# My Task Manager Website

A simple task manager built with React. This project allows users to create, track, and manage tasks efficiently.

## Features

- Add, edit, and delete tasks
- Mark tasks as complete or pending
- Lightweight and fast
- Uses icons from `lucide-react`

## Technologies Used

- React 18
- React DOM
- Lucide React (for icons)
- Vite (for development and build)

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm
  
### Installation

1. Clone the repository:
```bash
git clone https://github.com/hayabusagsxr/my-task-manager-website.git
```
2. Navigate to the project folder:
```bash
cd my-task-manager-website
```  
4. Install dependencies:
```bash
npm install
```
5. Start the development server:
```bash
npm run dev
```
